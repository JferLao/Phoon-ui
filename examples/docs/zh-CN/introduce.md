## Phoon UI

Phoon UI,是一套基于 Vue.js 的开源 UI 组件库,主要服务于 PC 界面的中后台产品。

### 特性

- 开箱即用的 Vue 组件
- 设计风格参考主流的组件库
- 友好的 API ，自由灵活地使用空间

### 支持环境

- 现代浏览器和 IE9 及以上（需要 polyfills）
